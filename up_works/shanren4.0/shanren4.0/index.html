关于我们

山人网络科技有限公司（SHANREN NETWORK AND TECHNOLOGY CO.,LTD），是一家专注于网络安全、大数据解决方案和软件系统研发的技术型互联山人网络科技有限公司（S，是一家专注于网的技术型互联网企业。公司业务方向涉及云计算、大数据、信息安全等领域，业务领域涵盖网络科技、计算机科技领域内的、技术转让，企业管理咨询，计算机软件开发等多种IT服务业务。

职位招聘

  • PHP程序员

    岗位职责：
    1、精通PHP脚本开发，有独立开发网站的能力，快速编写结构清晰、代码优美整洁的类库，有自己版权代码库者优先；
    2、注重程序的安全性，能够在程序开发初期通过设计模式避免未来可能出现的问题；
    3、可进行主流开源php程序的二次开发、模板改进和一定的系统结构修改能力；
    4、熟悉XML、Javascript、AJAX等web技术; 熟练掌mysql数据库应用；
    5、负责公司的开源电商产品部分模块开发，协调架构师进行编程工作。

    IOS/Android APP开发工程师

    岗位职责：
    1、从事(IOS/Android)网站系统延伸移动端开发、维护、管理工作，完成符合功能、性能要求的APP
    软件产品。 2．讨论并充分理解需要业务，按照需求业务完成APP终端开发工作；
    3．根据公司开源产品所分配到的任务，按时按质量完成整体设计；
    4．编写规范的技术文档， 编制项目文档、记录质量测试结果；
    5．根据需求不断修改完善终端系统； 研究用户体验的技术细节；
    6．独立完成APP程序的稳定性测试，兼容多个手机版本等测试等。

    Java工程师

    岗位职责：
    1、完成软件系统代码的实现，编写代码注释和开发文档；
    2、辅助进行系统的功能定义,程序设计；
    3、根据设计文档或需求说明完成代码编写，调试，测试和维护；
    4、分析并解决软件开发过程中的问题；
    5、配合项目经理完成相关任务目标。
    任职资格：
    1、计算机或相关专业本科学历以上；
    2、具有一定java基础
    3、有一定 web mvc 编程基础；
    4、熟练使用mysql；
    5、吃苦耐劳，有较强的团队协作能力。
